Requirements: Java, Junit 3

About the upload:
1. ./lockfree_skiplist/src/lockfree_skiplist/node.java: Skiplist node class. Contains setters and getters for node data.
2. ./lockfree_skiplist/src/lockfree_skiplist/skiplist.java: Skiplist class. Contains insert,remove, print, find and search methods of skiplist.
3. ./lockfree_skiplist/src/lockfree_skiplist/skiplisttest.java: Driver for skiplist class. Contains functions for sanity check and performance evaluation of the skiplist.
4. ./lockfree_skiplist/src/lockfree_skiplist/test.java: Contains main function.
